# Medical_Service_App

**Medical_Service_App** is a desktop application developed using **Visual Basic .NET (VB.NET)**. This project is designed to facilitate medical service management, focusing on administrative workflows and record organization within a .NET environment.

## 🛠️ Technologies & Tools

* **Language:** Visual Basic .NET (VB.NET).
* **Framework:** .NET Framework (Windows Forms / Desktop).
* **IDE:** Microsoft Visual Studio.
* **Version Control:** Git (optimized with `.gitignore` and `.gitattributes`).

## 📂 Repository Structure

Based on the current repository layout:

* **`/Proyecto`**: Contains the source code, including application logic and user interface forms.
* **`/Recursos`**: Dedicated folder for visual assets, icons, and external resources.
* **`Proyecto.sln`**: The Visual Studio Solution file for managing the entire project.
* **`ReadMe.txt`**: Additional internal documentation and legacy notes.
